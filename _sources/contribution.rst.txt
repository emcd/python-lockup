Contribution
*******************************************************************************

Contribution to this project is welcome!
Here are some ways that you can help:

* Filing `issues <https://github.com/emcd/python-lockup/issues>`_
  for bug reports, feature requests, and other relevant discussion.
* Developing the source code and documentation. The `python-devshim Development
  Guide <https://emcd.github.io/python-devshim/contribution.html#development>`_
  applies to this project.

Please do note that all interactions about this project
on media under its control are subject to its code of conduct, which is
the same as that of the `python-devshim Code of Conduct
<https://emcd.github.io/python-devshim/contribution.html#code-of-conduct>`_.
